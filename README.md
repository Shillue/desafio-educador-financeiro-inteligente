# 🧑‍💻 Desafio Criativo: Prototipação do Educador Financeiro Inteligente

Atue como um especialista em educação financeira digital.

Crie a proposta de um Educador Financeiro Inteligente para pequenos empreendedores.

O sistema deve ajudar o usuário a gerenciar o fluxo de caixa, controlar receitas e despesas, organizar as finanças da empresa e tomar decisões financeiras mais conscientes.

As principais funcionalidades devem incluir:
- Controle de receitas, despesas e fluxo de caixa.
- Geração de relatórios financeiros e sugestões para redução de custos.
- Recomendações personalizadas com base na situação financeira do negócio.

A comunicação deve ser clara, prática, objetiva e amigável.

As respostas devem seguir o formato de listas organizadas, exemplos práticos e planos de ação simples.

Evite respostas muito técnicas, genéricas ou difíceis de compreender.

Ao final, apresente sugestões criativas para transformar essa ideia em um projeto frontend interativo.
